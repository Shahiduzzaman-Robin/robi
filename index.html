MTD / CTD / YTD / YOY / MOM / DOD / D2D Metrics

English:

  • MTD (Month-to-Date): Performance from the start of the current month to today.
  • CTD (Cycle-to-Date): Performance from the start of a defined cycle (business-defined period).
  • YTD (Year-to-Date): Performance from the start of the year to today.
  • YOY (Year-over-Year): Comparison with the same period last year.
  • MOM (Month-over-Month): Comparison between the current month and previous month.
  • DOD (Day-over-Day): Comparison between consecutive days.
  • D2D (Day-to-Day): Daily movement/trend tracking (similar to DOD but more trend-focused).

Bangla:

  • MTD (Month-to-Date): চলতি মাসের শুরু থেকে আজ পর্যন্ত পারফরম্যান্স।
  • CTD (Cycle-to-Date): নির্ধারিত সাইকেল শুরু থেকে বর্তমান পর্যন্ত পারফরম্যান্স।
  • YTD (Year-to-Date): বছরের শুরু থেকে আজ পর্যন্ত পারফরম্যান্স।
  • YOY (Year-over-Year): গত বছরের একই সময়ের সাথে তুলনা।
  • MOM (Month-over-Month): বর্তমান মাস বনাম আগের মাসের তুলনা।
  • DOD (Day-over-Day): এক দিনের সাথে পরের দিনের তুলনা।
  • D2D (Day-to-Day): দৈনিক পরিবর্তন বা ট্রেন্ড ট্র্যাকিং (DOD-এর মতো, তবে ট্রেন্ড-ফোকাসড)।
